Project Structure

The structure of the ARQUITO HTML template:

  • js - all common scripts used on the pages, theme scripts included in theme.js file
  • css - all common scripts used on the pages, theme styles included in theme.css, responsive.css and dark.css files
  • img - images, icons and backgrounds for the project
  • fonts - all icons used on the pages, included Linearicons and Icofont files
  • img - photo images for the project
